# 💳 Fraud Detection using SQL

## 📌 Project Overview
This project analyzes credit card transaction data to identify fraudulent patterns using SQL and basic data preprocessing techniques.

---

## 🎯 Objectives
- Detect fraudulent transactions
- Analyze transaction behavior
- Identify suspicious patterns
- Categorize transactions based on amount

---

## 🛠 Tools & Technologies
- MySQL
- SQL (Aggregation, CASE, Window Functions)
- Python (Pandas)

---

## 📊 Key Insights
- Fraud transactions are very rare (imbalanced dataset)
- Small transactions are often used to test stolen cards
- High-value transactions carry higher fraud risk
- Categorization helps identify suspicious behavior patterns

---

## 🔍 SQL Techniques Used
- GROUP BY & Aggregations
- CASE Statements
- Window Functions (RANK)
- Filtering & Sorting

---

## 📁 Project Structure
- data/ → Dataset
- sql/ → SQL queries
- notebooks/ → Python analysis

---

## 🚀 Outcome
Developed a fraud detection analysis system using SQL to identify suspicious transaction patterns and generate meaningful insights for decision-making.
